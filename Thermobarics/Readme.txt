Thermobarics

By: UFIOES

To install, extract Thermobaricsinto \Steam\SteamApps\common\TUG\Mods\
Then add "Mods/Thermobarics" to \Steam\SteamApps\common\TUG\Config\mods.txt
e.g.

Mods
{
	"Mods/Thermobarics"
	"Game/Dev"
	"Game/Survival"
	"Game/Creative"
	"Game/Core"
}
